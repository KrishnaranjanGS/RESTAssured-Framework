# RESTAssured-Framework
A basic REST Assured automation framework with Reqres api as example 

**Overview**
This project is a REST API automation framework built using Rest Assured, TestNG, and Java.
It follows a basic Page Object Model (POM)-style structure for API testing.

**Tech Stack**
Java
REST Assured
TestNG
Maven

**Project Structure**
src/main/java     → API layer, utilities, constants  
src/test/java     → Test classes  
src/main/resources → Config files

**Configuration**
Environment config is maintained in:
src/main/resources/config/config.properties
Example:
base.uri=https://reqres.in
api.key=xxxxxxx
